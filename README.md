PonyMemoGL
==========

Memory Card Game in OpenGL [Dumb school project]


Prerequisite
============

GLUT or freeGLUT and of course g++

For debian users:
- sudo apt-get install mesa-common-dev freeglut3-dev


Compilation
===========
Still for linux users:
g++ PonyMemoGL.cpp -o PonyMemoGL.o -lglut -lGLU
